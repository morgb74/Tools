# Tools

This repository contains a collection of custom and miscellaneous Python tools, scripts, and utilities for automating practical tasks, processing data, archiving content, and supporting day-to-day workflows. The tools are maintained as standalone helpers and may vary in scope, maturity, and purpose depending on the problem they were created to solve.

## Repository Structure

```text
Tools/
├── docs/
│   └── Overview.md
├── src/
│   └── forexfactory_crawler.py
├── test/
│   └── Readme.md
└── README.md
```

- `docs/` contains user guides and supporting documentation.
- `src/` contains the Python source files and standalone utility scripts.
- `test/` is reserved for future test files, fixtures, and validation scripts.
- `README.md` provides the top-level repository overview.
